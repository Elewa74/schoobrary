Schoobrary · Brand Style Guide

Color Palette & Visual Identity System

A complete reference for Schoobrary's design tokens — every color, gradient, shadow, radius, and typography choice used on schoobrary_v3.html, ready to drop into another website.

All values are taken from the live :root tokens in the source file. CSS variable names are preserved, so this palette can be pasted into any project unchanged.

1. Primary — Blue (Brand Anchor)

Schoobrary's identity is built on a confident, education-friendly blue. Use it for the logo, primary navigation states, links, hero backgrounds, and the dominant call-to-action gradients.

5. Background Styles

Five canonical surface treatments cover every section on the site.

Default page surface — White / Off-White

background: var(--white) for content cards, var(--off-white) for the page body. Pair with --text typography.

Soft section band — Blue Pale

background: var(--blue-pale) separates feature blocks without breaking the calm tone. Use with the --border hairline.

Neutral utility band — Grey

background: var(--grey) for logo strips, code panels, and tertiary surfaces.

Hero gradient — Deep blue diagonal

linear-gradient(145deg, var(--blue-dark) 0%, var(--blue-mid) 40%, var(--blue) 100%) — overlay with a 32 px radial dot grid at rgba(255,255,255,.12) and two ambient glows (green at 18 % & orange at 12 %).

CTA section gradient

Same family as the hero (145deg, --blue-dark → --blue-mid → --blue) with two soft radial wash overlays in green and orange.

Footer — Near-black navy

background:#061428 with body text at rgba(255,255,255,.4) and copyright at rgba(255,255,255,.25).

8. Typography

Two typefaces from the IBM Plex super-family, both loaded from Google Fonts. IBM Plex Sans brings clean, technical confidence to every heading; IBM Plex Serif carries body and reading text with warm, editorial readability.

IBM Plex Sans · 700
Every book, every lesson
IBM Plex Serif · 400 — the body family for paragraphs and reading. Set at 17px / 1.65 for body copy, 13–15px for UI controls. The serif lends quiet authority to long-form content while Plex Sans carries headings, eyebrows, and the wordmark.

13. Copy & Paste — Full :root Token Block

Drop this directly into the top of any stylesheet to inherit the entire Schoobrary system.

/* Schoobrary brand tokens */
:root {
  /* Brand palette */
  --blue:        #1E88E5;
  --blue-mid:    #1565C0;
  --blue-dark:   #0D3B7A;
  --blue-light:  #E3F2FD;
  --blue-pale:   #F0F7FF;

  --green:       #4DB848;
  --green-dark:  #2E7D32;
  --green-light: #E8F5E9;
  --green-pale:  #F1FBF1;

  --orange:      #F05A28;
  --orange-mid:  #D94D1E;
  --orange-light:#FFF0EB;
  --orange-pale: #FFF8F5;

  --navy:        #0D2240;
  --navy-mid:    #1A3460;
  --white:       #FFFFFF;
  --off-white:   #F8FBFF;
  --grey:        #F2F5F9;
  --border:      #D8E6F5;
  --text:        #1A2B42;
  --muted:      #5A6E88;

  /* Elevation */
  --shadow-sm:  0 2px 8px rgba(30,136,229,.08);
  --shadow-md:  0 8px 28px rgba(30,136,229,.12);
  --shadow-lg:  0 20px 56px rgba(13,34,64,.14);

  /* Radius */
  --r:           12px;
  --r-lg:       20px;
  --r-xl:       28px;
}

/* Body defaults — reading set in Plex Serif */
body {
  font-family: 'IBM Plex Serif', serif;
  background:  var(--white);
  color:       var(--text);
  line-height: 1.65;
}

/* Headings — Plex Sans */
h1, h2, h3, h4, h5, h6, .eyebrow, .btn, .logo-text {
  font-family: 'IBM Plex Sans', sans-serif;
}

/* Hero / CTA gradient */
.hero {
  background: linear-gradient(145deg,
                var(--blue-dark) 0%,
                var(--blue-mid) 40%,
                var(--blue) 100%);
}

/* Primary CTA */
.btn-orange {
  background: linear-gradient(135deg, var(--orange), #e04820);
  color:      #fff;
  box-shadow: 0 4px 18px rgba(240,90,40,.35);
}
